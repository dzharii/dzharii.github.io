January 17, 2026

Interactive Tools and Apps‚Äã

https://dustinbrett.com - Full desktop-environment-in-the-browser experience (daedalOS) where you can explore a faux OS, launch apps, and get lost in a surprisingly complete interaction model. Not just a theme but a functional environment with real heft behind it.

https://github.com/DustinBrett/daedalOS - Source and feature map for daedalOS. Concrete, scannable README that reveals the real scope (apps, emulators, integrations like pinball, Quake, etc.).

https://garden.simonsarris.com - Interactive garden designer where you create and manipulate plants/objects directly with a canvas-like workflow. Genuinely playful, craft-forward tool rather than a static portfolio page.

https://www.masswerk.at/pet - Surprisingly capable Commodore PET 2001 emulator in the browser, more usable than many "toy" emulators and backed by a larger retro-computing archive.

https://creatura.zweibel-cocaine.com - Quirky interactive WebGL canvas experiment powered by Zig via WASM where you move the mouse and click to explore variations. Compact demo that showcases modern web graphics and non-JS toolchains.

https://carefulwords.com - Fast, minimal word-exploration tool that is intentionally more than a thesaurus. Makes language browsing feel like an interactive instrument.

https://webring.theoldnet.com - Living WebRing with a frameset browser and a "random site" flow. Curated-ish old-web exploration as an experience.

https://nathanfriend.com - Standout "personal site as a toolbox" with many small interactive projects and utilities that are actually fun to use, presented cleanly and discoverably.

https://ka.ge - Surprisingly faithful Windows 98 recreation in the browser, with a bunch of built-in faux apps (Notepad, Paint, Winamp, etc.).

https://ben.gy - Fully interactive, retro Macintosh-style desktop that doubles as a personal site, complete with faux menus and multiple playable mini-apps and games.

https://keepitwiel.github.io/erosion-sim - In-browser hydraulic erosion simulator where you can tweak parameters and watch terrain evolve.

https://wikilangs.org - Curated set of language playgrounds (interactive demos per language) that makes it easy to try unusual or niche languages without setup.

https://pdfix.org - Local-first, in-browser PDF editor aimed at privacy, where PDFs are processed on your device rather than uploaded.

https://marscalendar.space - Niche but well-executed little web app that gives you a Martian calendar/time view and links out to related space exploration projects.

https://searchmysite.net - Search engine specifically for personal and independent sites, with an open-source angle.

https://tuvix.app - Open-source, self-hostable RSS reader positioned hard against algorithmic feeds.

https://tylervigen.com/spurious-correlations - Canonical spurious correlations playground with depth behind the jokes. Each correlation expands into sources and reproducible Python.

https://tools.ddanieltan.dev - Small collection of lightweight, single-purpose web tools that load quickly and do one job.

https://www.autostrum.com - Guitar tab creation and playback web app with unusually "product-grade" feature set for a personal project.

Deep Technical Writing‚Äã

https://danilafe.com - Compiler and PL-heavy blog with real depth (formal verification, type systems, serious engineering writeups). Long, structured posts with reading-time estimates and good navigation.

https://finnvolkel.com - Unusually deep, research-adjacent writing about algorithms and incremental computation. Posts read like a careful notebook with concrete derivations.

https://venam.net - Rare "systems + philosophy" blog that delivers on the systems side with well-researched Unix/Linux deep dives (Linux input stack, PipeWire internals).

https://blog.domainmess.org - Compact goldmine of practical Linux/network/storage posts (eBPF traffic distribution, block IO debugging, mkosi, virtiofs, routing).

https://dave.engineer - Personal site that is also a curated engineering artifact with an interactive terminal UI on the homepage plus genuinely useful subprojects.

https://www.masswerk.at/6502/6502_instruction_set.html - Dense, utilitarian 6502 instruction set reference that people actually keep open while building emulators.

https://www.masswerk.at/6502/assembler.html - Practical companion reference around 6502 assembly/assemblers. Integrated reference set.

https://xorvoid.com - Serious engineer's blog with unusually deep, technical work: low-level systems topics, reverse engineering, decompiler/compiler toys, and math-forward series on finite fields.

https://yyhh.org - Deep database and systems performance writing centered on building Datalevin, with posts explicitly tackling throughput/latency and benchmark-driven claims.

https://dmitry.gr - Retrocomputing and hardware-reverse-engineering project log with multiple "how is this even possible" builds (Linux on Intel 4004).

https://serhack.me - Security research blog with real reverse-engineering and malware/firmware analysis.

https://xlii.space - Compact blog index with posts on allocators, Haskell, CUE, and grumpy workflow takes.

https://lik.ai - Solid engineering blog with concrete benchmark-style posts (SQLite ID benchmarks across UUID variants) and comprehensive Markdown reference.

https://bernsteinbear.com - Programming-languages-and-compilers-heavy blog with practical PL work (interpreters, JITs, runtimes).

https://web.xiaolong-hosting.com - Unusually dense personal site that reads like a real engineering notebook with lots of self-contained writeups on Scheme/functional programming.

https://blog.winricklabs.com - Technical blog with genuinely "I built or reverse-engineered something" energy (security/RE/hardware topics).

https://vincent.bernat.ch - Deep systems/networking content with a "reference manual" feel with diagrams and config detail.

https://p.migdal.pl - Strong technical writing and research-adjacent work. Structured around essays, projects, and publications.

https://matt.might.net/articles - High-density archive of computer-science writing (PL, compilers, systems, academic-but-readable explainers).

https://gabrielgambetta.com - Concentrated set of high-quality technical essays (computer graphics "from scratch" style explanations).

https://tomverbeure.github.io - Deeply technical electronics blog with long, careful writeups (FPGA work, vintage test equipment) with real measurements and teardowns.

https://wcampbell.dev - Unusually dense, low-level systems writing (ELF internals, compiler behavior, binary formats).

https://konstantin.antselovich.com - Battle-tested performance and database notebook with standout post on pushing MySQL connection counts on Linux.

https://abstractnonsense.xyz - Math and CS-heavy personal blog that actually ships substance (implementing eigenvalue solver in WebAssembly).

https://wheybags.com/blog/emperor.html - Unusually thorough reverse-engineering and exploitation writeup with concrete assembly-level detail.

https://royalicing.com - Strong engineering-and-product blog with standout WebAssembly track explaining Elixir DSL for wasm compilation.

https://skushagra.com - Sharp engineering blog with real specifics on SwiftUI/AppKit implementation.
